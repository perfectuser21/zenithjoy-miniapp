This directory contains generated React/Tailwind/Next.js code for all currently selected frames from `designs/pencil-ui.pen`.

Files:
- `page.tsx`: a single App Router page that renders all selected mobile screens in a gallery.

Suggested usage:
1. Copy `page.tsx` into a Next.js `app/.../page.tsx` route.
2. If needed, split individual screen functions into separate files.
3. Replace the inline mock content with live data or CMS bindings.

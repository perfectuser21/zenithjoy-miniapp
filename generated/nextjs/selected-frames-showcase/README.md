# Selected Frames Showcase

This folder contains a multi-file Next.js-style output for the currently selected Pencil frames.

## Structure

- `app/page.tsx`: example App Router entry page
- `components/SelectedFramesShowcase.tsx`: main interactive shell
- `components/showcase-data.ts`: page ids, navigation groups, demo content
- `components/showcase-primitives.tsx`: shared layout primitives
- `components/showcase-pages.tsx`: individual mobile page implementations
- `../SelectedFramesShowcase.tsx`: original single-file version kept for convenience

## Usage

If you want to drop this into a Next.js app, copy:

- `app/page.tsx`
- `components/SelectedFramesShowcase.tsx`
- `components/showcase-data.ts`
- `components/showcase-primitives.tsx`
- `components/showcase-pages.tsx`

The single-file `../SelectedFramesShowcase.tsx` remains available if you prefer a one-file handoff instead of the modular version.
